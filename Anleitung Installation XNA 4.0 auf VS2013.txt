Vor der Installation von XNA muss Visual Studio 2013 (z.b. die Premium Version) von DreamSpark heruntergeladen und installiert werden.

XNA Setup
---------

1. Download xnags40_setup.exe and save it to your hard drive

   http://www.microsoft.com/downloads/en/details.aspx?FamilyID=9ac86eca-206f-4274-97f2-ef6c8b1f478f
    
2. Run xnags40_setup.exe /x to extract the contents to a folder of your choosing

3. Go to the folder you extracted to in step 2 and run the MSI named redists.msi

   Note: This MSI will only display a small progress bar while it is installing and will not tell you when it is done.  When the progress bar disappears, continue to the next step.

3. Run the MSI at %ProgramFiles%\Microsoft XNA\XNA Game Studio\v4.0\Setup\XLiveRedist.msi
4. Run the MSI at %ProgramFiles%\Microsoft XNA\XNA Game Studio\v4.0\Redist\XNA FX Redist\xnafx40_redist.msi
5. Run the MSI at %ProgramFiles%\Microsoft XNA\XNA Game Studio\v4.0\Setup\xnaliveproxy.msi
6. Run the MSI at %ProgramFiles%\Microsoft XNA\XNA Game Studio\v4.0\Setup\xnags_platform_tools.msi
7. Run the MSI at %ProgramFiles%\Microsoft XNA\XNA Game Studio\v4.0\Setup\xnags_shared.msi

8. Die MSI unter %ProgramFiles%\Microsoft XNA\XNA Game Studio\v4.0\Setup\xnags_visualstudio.msi
   kann VS 2013 nicht erkennen, weil sie zu alt ist, daher muss man die Dateien manuell entpacken:
   
   msiexec /a xnags_visualstudio.msi /qb TARGETDIR=C:\Beispielordner
   
   Die Ordner, die dann in C:\Beispielordner sind, müssen nun in den Plugin-Ordner von Visual Studio 2013
   kopiert werden:
   C:\Program Files (x86)\Microsoft Visual Studio 12.0\Common7\IDE
   
   Danach sollte sich der XNA-Plugin-Ordner hier befinden:
   
   C:\Program Files (x86)\Microsoft Visual Studio 12.0\Common7\IDE\Extensions\Microsoft\XNA Game Studio 4.0
   
9. Öffne die Datei "extension.vsixmanifest" im XNA-Pluginordner und tausche die folgende Zeile aus:

   <VisualStudio Version="10.0">
   
   =>
   
   <VisualStudio Version="12.0">
   
   (Version 10.0 ist VS 2010, die aktuelle Version ist 12.0 aka VS 2013)
   
10. Extension-Cache rebuilden:

   "C:\Program Files (x86)\Microsoft Visual Studio 12.0\Common7\IDE\devenv.exe" /setup
             
11. Go to the folder you extracted to in step 2 and run the MSI named arpentry.msi
